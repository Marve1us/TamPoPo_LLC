Product Requirements Document (PRD)

Product Name:
TamPoPo LLC – Texas AI Solutions Website

Prepared For: Replit Web Build
Document Version: v1.1 (Updated Company Name)

1) Vision Statement

Vision:
To become the most trusted AI solutions partner for small businesses in Texas by delivering affordable, easy-to-use automation tools that help companies grow faster, operate smarter, and convert more customers online.

2) Mission Statement

Mission:
TamPoPo LLC provides Texas AI solutions that empower local businesses with modern AI-driven websites, CRM systems, automated voice and chat assistants, and SEO optimization—without technical complexity—so small businesses can compete with large enterprises.

3) Product Summary

A clean, futuristic marketing website that showcases AI services for small businesses. The site will allow potential clients to schedule demos, view service offerings, and allow administrators to update content, pricing, and images through a secure admin portal.

The site must be responsive, secure, and designed with a cyberpunk aesthetic (teal + silver).

10) Security Requirements (Secure Coding Practices)
Frontend

HTTPS only

Input validation on all forms

CAPTCHA on contact/demo forms

Backend

Password hashing (Supabase Auth or bcrypt)

Role-based access control (admin only)

No API keys exposed in frontend

Environment variables stored in Replit Secrets

General

Content Security Policy (CSP)

Rate limiting on admin login

Audit logs for admin changes
